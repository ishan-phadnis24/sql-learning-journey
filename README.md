# Data Analytics Portfolio

This repository documents my hands-on learning journey in SQL and data analytics using Google BigQuery. Through a collection of real-world datasets, I practice solving business problems, exploring data, and generating insights using analytical SQL techniques.

The portfolio includes datasets from multiple domains:

* **Flipkart** – E-commerce analytics involving customers, orders, products, and returns.
* **Rapido** – Transportation and mobility analytics focused on ride operations and customer trends.
* **Apollo Hospitals** – Healthcare analytics involving operational and patient-related data.
* **Dream11** – Fantasy sports analytics exploring user engagement and platform activity.
* **Global Test Solutions** – Human Resources analytics using employee, department, job, job history, location, region, and country data to analyze workforce structure and organizational performance.

## Skills Practiced

* SQL
* Google BigQuery
* Joins
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* ROW_NUMBER(), RANK(), DENSE_RANK()
* Running Totals and Moving Aggregations
* Data Exploration
* Business Problem Solving

This portfolio reflects my ability to work with diverse datasets and apply SQL to answer business questions across e-commerce, healthcare, transportation, sports, and human resources domains.
